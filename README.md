# 🔐 Network Intrusion Detection System using Suricata

## 📌 Overview
This project demonstrates the implementation of a **Network Intrusion Detection System (NIDS)** using **Suricata** on **Kali Linux**.  
The system monitors network traffic in real-time and detects suspicious activity using custom-defined rules.

---

## 🛠️ Tools & Technologies
- Suricata (IDS Engine)
- Kali Linux
- Nmap 
- Linux Terminal
